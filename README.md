# Lawyer Landing Page

Современный сайт для юридической компании или частного юриста, созданный по дизайну из Figma.  
Проект выполнен в минималистичном и профессиональном стиле с акцентом на доверие, удобство и понятную структуру.

## О проекте

Главная задача сайта — представить юридические услуги, рассказать о компании и помочь пользователю быстро связаться со специалистом.

Дизайн сочетает в себе строгий стиль, современный интерфейс и адаптивную верстку для всех устройств.

## Главная секция

В первом экране размещён основной оффер компании, краткое описание услуг и кнопка для связи или консультации.  
Секция сделана так, чтобы сразу привлечь внимание пользователя и показать основные преимущества.

## О компании

Блок рассказывает о компании, опыте работы и профессиональном подходе.  
Здесь размещается краткая информация о юристах и деятельности организации.

## Услуги

Секция с перечнем юридических услуг.  
Каждая карточка содержит название услуги и короткое описание, чтобы пользователь мог быстро понять направление работы компании.

## Преимущества

Блок с основными преимуществами компании:

- опыт работы
- профессиональная команда
- индивидуальный подход
- конфиденциальность
- поддержка клиентов

## Отзывы

Секция с отзывами клиентов помогает повысить доверие к компании и показывает опыт работы с реальными клиентами.

## Контакты

В нижней части сайта размещены контакты, адрес, номер телефона и форма обратной связи для быстрой консультации.
